An Open Multimodal Sensing Dataset for Wireless Communication and Cooperative Perception

What is Multimodal-Wireless?

An open-source multimodal sensing dataset designed for communication and cooperative perception research.
Generated through an integrated and customizable data pipeline built upon the CARLA simulator and Sionna framework, the dataset features:

  • ✦ 161,400 frames collected across 4 virtual towns, 16 communication scenarios, and 3 weather conditions, encompassing multiple sensing modalitiesâ€”communication channel, LiDAR, RGB and depth cameras, IMU, and radar.
  • ✦ 100 Hz sampled channel for diverse applications, along with 100 Hz oversampled, temporal aligned sensor data which can be easily downsampled for practical usage.
  • ✦ All-weather resilience by design.
  • ✦ Full flexibility and extensibility by simply modifying a configuration file.
  • ✦ A unified resource for vehicle-to-everything (V2X) communication and perception.
