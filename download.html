Download the Multimodal-Wireless Dataset

The Multimodal-Wireless dataset is available for download. Please choose the source that best suits you:

  • Global (OneDrive): Download from OneDrive
  • China mainland mirror: Download from Quark
Tips:
  • The dataset has been updated in Nov. 17, 2025. Please download the the latest version.
  • The channel data has been augmented with additional AoD and AoA values based on the local coordinate system of planar arrays in Nov. 30, 2025. Detailed specifications of the channel data can be found at Data Collection.
  • The sunny/rainy folder contains the dataset collected under sunny/rainy conditions, in which the channel data folder contains channel data under various settings and the sensor data folder contains sensor data of the scenario. Users are free to combine the suitable channel data with the sensor data for their own research.
  • The Blender_Towns folder provides Blender-format mirrors of all CARLA towns for advanced scenario replay and visualization.
  • The foggy dataset is uploading. Stay tuned for updates!
